# Test CleverIT
Pequeña API que obtiene los números primos ordenados en forma descendente.
## Dependencias
- ExpressJS
- Swagger UI Express
- Swagger JSDoc
- Jest
- Supertest
- Typescript
- Nodemon
- Dotenv

## Para ejecutar en entorno dev
``npm run dev``

## Para crear build
``npm run build``

## Para ejecutar test
``npm run test``

## Ver documentación Swagger
Ir a http://localhost:<puerto>/docs
